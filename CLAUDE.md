# Fulfilment WMS Customer Portal

## Project
Frontend portal for small clients. React + TypeScript + Vite.

## API
- OpenAPI spec: ./docs/openapi.yaml
- Base URL: https://lpw.betta.ru:8084/grh/api
- Auth: Basic Authentication

## Stack
- React 18 + TypeScript
- Tailwind CSS
- React Query (для API calls)
- React Router v6

## Key entities
- Indocs (Входящие документы)
- Outdocs (Исходящие документы)
- Orders (Заказы)
- Inventory (остатки)
